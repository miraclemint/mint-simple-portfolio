# Mint Tharin Portfolio

Welcome to the personal portfolio website of **Mint Tharin**, a One Person Business Analyst. This site showcases key professional projects and provides contact details.

## Features
- **Modern Dark Theme:** Sleek, high-contrast dark mode as default.
- **Light Theme Toggle:** Easily switch to a light theme for better readability.
- **Responsive Design:** Optimized for mobile and desktop.
- **Tech Stack:**
  - HTML5
  - CSS3 (with custom variables)
  - Vanilla JavaScript

## Project Structure
- `docs/index.html`: Main structure of the site.
- `docs/assets/css/style.css`: Theming and layout styles.
- `docs/assets/js/main.js`: Interactivity logic (theme toggle).
- `docs/assets/images/`: Project thumbnails and images.
