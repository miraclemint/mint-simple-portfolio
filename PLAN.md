# Portfolio Plan: Mint Tharin (One Person Business)

## Objective
Build a clean, modern, dark-themed portfolio website for "Mint Tharin" to showcase Business Analyst projects and provide contact details, hosted on GitHub Pages.

## Design Specifications
*   **Theme:** Modern Dark Theme (using dark grays/blacks and a high-contrast accent color).
*   **Typography:** [JetBrains Mono](https://www.jetbrains.com/lp/mono/) (Google Fonts).
*   **Structure:** Single-page layout (responsive).

## Implementation Plan

### Phase 1: Setup
1.  Initialize a new GitHub repository named `mint-tharin.github.io`.
2.  Clone the repository to local development.
3.  Set up the base folder structure:
    *   `index.html` (Main structure)
    *   `assets/css/style.css` (Theming & Layout)
    *   `assets/js/main.js` (Interactivity)
    *   `assets/images/` (Project thumbnails)

### Phase 2: Core Development
1.  **HTML Structure:**
    *   `<header>`: Navigation.
    *   `<section id="hero">`: "Mint Tharin" + Tagline (One Person Business).
    *   `<section id="projects">`: Grid layout for 3 featured projects.
    *   `<section id="contact">`: Contact details/form.
2.  **Styling (CSS):**
    *   Apply JetBrains Mono via Google Fonts.
    *   Implement dark mode palette (e.g., Background: `#1e1e1e`, Text: `#e0e0e0`).
    *   Add responsive grid for project cards.
3.  **Content:**
    *   Write the Hero section content.
    *   Draft descriptions and technical highlights for the 3 Business Analyst projects.
    *   Input contact details (LinkedIn, email).

### Phase 3: Deployment
1.  Verify the site locally.
2.  Push code to GitHub.
3.  Enable "GitHub Pages" in the repository Settings.
4.  Verify live site at `https://mint-tharin.github.io`.

## Verification
*   Check mobile responsiveness.
*   Validate font loading and dark theme consistency.
*   Confirm all links (projects/contact) work correctly.
